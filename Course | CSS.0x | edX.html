Welcome to the CSS Basics course!

Hello,

Thank you for registering to this course, where you will learn how to take your Web pages from bland to bold with the power of CSS styling. You will review historical and current design trends, and see how to apply the concepts in a beautiful way, use relative positioning, and much more. Bring your HTML knowledge, and plan to spend 5 to 7 hours on each of five modules, including lots of hands-on practice.

To get started, please read the "Course information" section, where you can become oriented to the course structure, the schedule, and how things will work in this course. Please take 5 mins to introduce yourself in the student introduction forum, and less than 5 minutes to answer our welcome survey :)

 

New feature: CSS Grid

We added two new pages at the end of module 5 (5.5). This is optional reading, meaning there are no exercises about them, as you might be interested in the very latest in CSS layout techniques, viz., the CSS Grid module. Since the summer of 2017, the major browsers support this module of CSS, which provides the beginnings of something professional designers have long been waiting for: a design grid.

It's been a few years now that the CSS Flexbox module gives advanced control over alignment and spacing of elements on a line or in a stack. The Grid module now allows alignment in two dimensions: to elements on the same line and to elements in the same column.

To learn about Flexbox, check out the "HTML5 and CSS Fundamentals course". Here, we only give some background and links. There is also a video with some of the people who created the CSS Grid module.

 

Course outline

  • The first set of lessons are available under "Module 1: Getting started with CSS", where we will show different examples of the amazing things you can do with CSS. This module includes a cameo of Bert Bos, co-inventor of CSS.
  • In Module 2, we will first review HTML and then learn how to build CSS rules with some basic properties that adjust your HTML document's color, font or spacing.
  • In Module 3, we are going to build slightly more complex Web pages using contextual selectors, IDs, classes and pseudo-classes. The idea is to combine some selectors together into one or more CSS rules. And thanks to the "cascading" part of Cascading Style Sheets, it won't be a problem to have multiple CSS rules competing for importance. Even with multiple rules that apply to the same HTML tag, the browser will know how to make it look!
  • Module 4 is about layout and positioning using CSS. You will start by understanding the box model - the foundation of layout on the Web. You will then learn about relative positioning and the float property in order to position your HTML elements in relation to one another on a Web page.

  • In Module 5 (last one!), the final course module, we are going to talk about the design aspect of Web pages and how to give your users the best experience! We will discuss lessons learned from historical Web design trends and then meet the newest fashions in Web design, to help you give your Web pages a modern look and feel. And we'll remind you not to forget accessibility and internationalization so as to accommodate an even wider, diverse audience.
